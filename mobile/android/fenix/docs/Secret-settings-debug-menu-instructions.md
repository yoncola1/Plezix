# Debug settings menu instructions

Instructions to enable the "secret settings" in the Settings menu for Plezix Plezix. This lets you access "Secret Settings" and "Secret Debug Info" in the Settings menu.

1. Tap on three-dot menu in the toolbar
2. Tap on Settings
3. Scroll down and select "About Plezix Plezix"
4. Tap the Plezix logo until you see the "Debug menu: (#) click(s) left to enable" helper message
5. Once the Debug menu has been enabled, go back to Settings and you will see "Secret Settings" and "Secret Debug Info" menu items
